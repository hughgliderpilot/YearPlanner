Instructions to make a Google Calendar public and find the Calendar ID

To make a calendar publicly viewable through yearplanner.org and to find the Calendar ID do the following:

 

1.       First set the Google calendar settings to make the calendar publicly viewable, see the following link:

https://support.google.com/calendar/answer/37083?hl=en

 

2.       The find the Calendar ID of the calendar you wish to view in yearplanner.org as follows:

a.       Go into your Google Calendar and select the gearwheel and then settings from the dropdown

 

b.       On the next screen click on the Calendars link

 

 

c.       Click on the calendar you want to set up on Yearplanner.org

 

d.       Copy the calendar id into https://yearplanner.org
